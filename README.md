# Damping analyzer #

## Getting Started ##

* Clone the repo
  ```bash
  git clone https://github.com/joshshep/damping_analyzer.git
  ```
* Install requirements
  ```bash
  cd damping_analyzer
  pip install -r requirements.txt
  ```
* Run a test
  ```bash
  python simple_superposition.py
  ```
